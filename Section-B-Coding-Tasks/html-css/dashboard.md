#  Product Dashboard

A modern and responsive **Product Dashboard UI** built using **HTML & CSS**.  
This dashboard layout includes a sidebar navigation menu, top navigation bar, and a product grid layout with interactive product cards.

---

## preview
![page](../../assets/dashboard.png)

##  Features

-  Sidebar Navigation (Dashboard, Products, Orders, Analytics, Settings)
-  Top Navbar with Search Bar
-  User Profile Section
-  Product Cards with:
  - Product Image
  - Product Name
  - Price
  - Stock Status (In Stock / Out of Stock)
-  Clean and Modern UI Design
-  Fully Responsive (Desktop, Tablet, Mobile)
-  Hover Effects on Cards
-  CSS Grid / Flexbox Layout

---

##  Layout Structure

### 1️ Sidebar
- Logo / Brand Name
- Navigation Links

### 2️ Top Navigation

- Product Dashboard Heading
